# hive

A Vue.js SPA sample application.

## Quick Start Guide

### Step 1: Clone Repo

```bash
git clone git@github.com:MeghanBomberger/hive.git
```

### Step 2: Install packages

```bash
cd hive
npm install
```

### Step 3: Start serve

```bash
npm serve
```

*project will be available on localhost:8080*
